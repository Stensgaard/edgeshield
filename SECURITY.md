# Security Policy

## Supported Versions

EdgeShield is currently under active development.

Security fixes are provided for the latest development branch and the latest stable release.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, email:

contact@hstensgaard.com

Include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested mitigation (optional)

We will acknowledge reports as soon as possible and work with you to resolve the issue before public disclosure.

Thank you for helping keep EdgeShield secure.
