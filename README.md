# EdgeShield

EdgeShield is a lightweight security gateway for web apps and APIs. It sits in front of an
upstream application, forwards safe traffic, and applies security policies such as request
inspection, rate limiting, API key checks, and authentication endpoint protection.

## Security

Security is a primary goal of EdgeShield.

If you discover a security vulnerability, please **do not** report it publicly through GitHub Issues. Instead, follow the instructions in [SECURITY.md](SECURITY.md) to report it privately.
